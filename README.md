# amarris-kitchen-backend

# Prerequisites
* .NET sdk 9 or 10
* Mysql server
* make sure 'amarris_kitchen' database exists on your Mysql server. check inside '/Sqlcode' if needed

# VSCODE
* Open project in Vscode.
* Configure the appsettings.json according to your MySQL connection (Port, User, and Password):
  * <img width="660" height="59" alt="image" src="https://github.com/user-attachments/assets/440d0a40-b50d-4b04-85fd-2cdd64899d6b"/> <br>
* Open vscode terminal
* enter dotnet restore:
  * <img width="624" height="109" alt="image" src="https://github.com/user-attachments/assets/9c897599-bd39-4aa9-9980-712ccee45b28" />
* enter dotnet run:
  * <img width="599" height="43" alt="image" src="https://github.com/user-attachments/assets/164d119e-281a-4d99-9d60-650eeb5f7856" />

* Use http://localhost:5250/ on browser to start the kiosk.
